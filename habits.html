Habits Colour Chart

Every day you've logged, colour-coded by habit:

  • green = positive habit (read, stretch, drink water)
  • yellow = neutral habit (intimacy, socialise)
  • red = negative habit (drink, self destructive, gamble)
  • grey = not logged that day

✏️ Adding or editing habits

Open js/data.js on GitHub and find the ACTIVITIES array. Each habit has:

  • key — internal id (no spaces), keep it the same once you start using it
  • label — the name shown on the site, safe to rename anytime
  • type — "positive", "neutral", or "negative", controls the colour

New habits will automatically appear here, in the Daily Entry checklist, and as dots on the Calendar.
