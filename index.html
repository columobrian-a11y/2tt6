    What is this site?

    2tt6 is my public accountability diary. Every entry, habit, and photo update here is edited directly on GitHub and auto-published through Netlify. Nothing fancy — just an old-web style logbook of where I'm at.

    Use the nav bar above to explore:

    • Affirmations — things I remind myself of.
    • Calendar — every diary entry, progress note, photo, food log, and habit in one place, by date.
    • Scoreboard — "days since..." counters.
    • Daily Entry — the form I fill out each day, which generates code to paste into the site's data file.
    • Habits Chart — a green/yellow/red colour chart of my habits over time.
